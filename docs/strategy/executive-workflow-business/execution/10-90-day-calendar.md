# 90-Day Execution Calendar

## Strategic Context

This calendar sequences all activities from the execution plan into a week-by-week schedule. It follows the three-phase structure from the marketing strategy:

- **Phase 1 (Weeks 1–4):** Foundation Lock — positioning, website, LinkedIn, first outbound
- **Phase 2 (Weeks 5–8):** Proof Expansion — content velocity, A/B testing, increased outbound
- **Phase 3 (Weeks 9–12):** Conversion Pressure — case studies, offer refinement, diagnostic pipeline

**Pattern applied:** `plan-comparison-methodology.md` — "Stabilize before expanding." Foundation activities must be complete before scaling content or outbound.

## Calendar Decision Rules

- If an activity is not producing results after 3 weeks, audit the approach before continuing
- If one channel dramatically outperforms others, reallocate time from underperformers
- If discovery calls start flowing, reduce content creation time and increase sales time
- Keep the weekly review sacred — it is the feedback loop that makes everything else work
